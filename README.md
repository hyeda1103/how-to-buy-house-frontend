## Getting Started

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/hyeda1103/how-to-buy-house-frontend.git
   ```
3. Install NPM packages
   ```sh
   yarn
   ```
4. Enter your API in `config.js`
   ```js
   const API_KEY = 'ENTER YOUR API';
   ```
